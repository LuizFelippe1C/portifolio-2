Jogo Quente ou Frio: Encontre o Número Secreto!

Este projeto é um jogo interativo de adivinhação, onde o objetivo é descobrir um número secreto gerado aleatoriamente pelo sistema. A dinâmica do jogo utiliza as pistas “quente” e “frio” para indicar se o palpite do jogador está próximo ou distante do número correto.

O jogo é desenvolvido com HTML, CSS e JavaScript. O HTML estrutura a página com um título chamativo, campos para inserir palpites e botões para interação. O CSS estiliza os elementos com cores vibrantes e feedback visual, criando uma atmosfera divertida e desafiadora.

O JavaScript é responsável pela lógica do jogo: ele gera o número secreto, compara os palpites dados, fornece as dicas e atualiza dinamicamente o conteúdo da página. Cada tentativa gera uma resposta imediata, tornando a experiência envolvente.

Esse tipo de jogo é excelente para quem está aprendendo programação, pois trabalha eventos, DOM, lógica condicional e loops.

Efeito Monalisa: Você Está Sendo Observado?

Este projeto é uma aplicação web que simula o famoso "Efeito Monalisa", no qual a imagem de uma figura parece seguir o olhar da pessoa observadora. O site utiliza HTML, CSS e JavaScript para proporcionar uma experiência visual instigante e envolvente.

O HTML estrutura a página com a imagem principal, elementos interativos e explicações sobre o fenômeno visual. O CSS cria um visual elegante e responsivo, enquanto o JavaScript rastreia a posição do cursor e atualiza o olhar da figura em tempo real.

Esse comportamento dinâmico cria a impressão de que a imagem está sempre observando o visitante. Além de entreter, o projeto propõe uma reflexão sobre percepção, arte e interação humano-computador.
