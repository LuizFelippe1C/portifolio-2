Jogo Quente ou Frio: Encontre o Número Secreto!

Este projeto é um jogo interativo de adivinhação, onde o objetivo é descobrir um número secreto gerado aleatoriamente pelo sistema. A dinâmica do jogo utiliza as pistas “quente” e “frio” para indicar se o palpite do jogador está próximo ou distante do número correto.

O jogo é desenvolvido com HTML, CSS e JavaScript. O HTML estrutura a página com um título chamativo, campos para inserir palpites e botões para interação. O CSS estiliza os elementos com cores vibrantes e feedback visual (como mudanças de cor conforme a proximidade), criando uma atmosfera divertida e desafiadora.

O JavaScript é responsável pela lógica do jogo: ele gera o número secreto, compara os palpites dados, fornece as dicas “quente”, “morno” ou “frio” e atualiza dinamicamente o conteúdo da página. Cada tentativa gera uma resposta imediata, tornando a experiência envolvente.

Esse tipo de jogo é excelente para quem está aprendendo programação, pois trabalha conceitos como eventos, manipulação do DOM, lógica condicional e loops. A combinação de design simples e mecânica envolvente torna o projeto acessível e empolgante para todos os públicos.

Efeito Monalisa: Você Está Sendo Observado?

Este projeto é uma aplicação web que simula o famoso "Efeito Monalisa", no qual a imagem de uma figura parece seguir o olhar da pessoa observadora, independentemente do ângulo. O site utiliza HTML, CSS e JavaScript para proporcionar uma experiência visual instigante e envolvente.

O HTML é responsável por estruturar o conteúdo da página, incluindo a imagem principal, elementos interativos e áreas de explicação sobre o fenômeno visual. O CSS é utilizado para criar um visual elegante e misterioso, com sombras, contrastes suaves e responsividade, garantindo que o efeito funcione bem em diferentes tamanhos de tela.

O JavaScript implementa a lógica do movimento dos olhos da figura, rastreando a posição do cursor e atualizando o olhar da imagem em tempo real. Esse comportamento dinâmico dá a impressão de que a figura está sempre observando quem acessa o site, criando um efeito hipnótico que desperta curiosidade e interesse.

Além de entreter, o projeto também propõe uma reflexão sobre percepção visual, arte e interação humano-computador. A escolha por usar apenas HTML, CSS e JavaScript demonstra como é possível criar efeitos visuais complexos e envolventes com tecnologias acessíveis, sem necessidade de plugins externos ou bibliotecas pesadas.
